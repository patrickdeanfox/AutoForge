# AutoForge Manifesto

AutoForge is designed to be an autonomous software development lifecycle (SDLC) engine. 
It aims to:
1. Orchestrate multiple specialized AI agents (Architect, Coder, Reviewer, Tester).
2. Integrate deeply with local tools (Claude CLI, Docker, Git).
3. Provide real-time observability into the agentic reasoning process.
4. Scale from simple scripts to complex microservices.
